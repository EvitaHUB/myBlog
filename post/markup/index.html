Annotation with Recogito

2020-12-18

Photo by Kelly Sikkema on Unsplash

Annotation with Recogito

For exploring this tool I searched for an interesting material to annotate. And I chose Livonian Chronicle of Henry a document in Latin describing historic events in Livonia (roughly corresponding to today’s inland Estonia and north of Latvia) and surrounding areas from 1180 to 1227.

It was written ca. 1229 by a priest named Henry (Latin: Henricus de Lettis). Apart from the few references in the Primary Chronicle compiled in Kievan Rus' in the twelfth century, it is the oldest known written document about the history of Estonia and Latvia.

I found the Latin text published in 1874 and downloaded pdf and txt files. Then uploaded text file to Recogito online annotation tool.

I annotated first 8 chapters of the chronical. And then checked the Map view. Even from the first few chapters we see that first missioners from Bremen area in Northen Germany are mentioned to come to what is now territory of Latvia and Estonia:

Also I downloaded CSV file and it contains nice string data. The problem with the text is, that it is not very clean - there are some machine reading errors, so same words can look a bit different in the text. Another problem with Latin is that its grammar inflections make many variations of the same words. In the picture below we see 3 variations of the same place name Turaida, but in the book it os at least 3 different.

Regarding geolocations - I like this thing very much, just I need to find how to add new locations, as I had several, that could not be added to my map. But I love visualizations and maps, so I will continue with them.
